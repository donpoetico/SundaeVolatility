# Change: Add UI/UX and Visual Design Specifications

## Why
To establish the visual and interaction design foundation for the project, ensuring all UI elements follow the diegetic philosophy and tactile standards defined in the UI/UX Visual Specification v1.0.

## What Changes
- Add `ui-diegetic` capability covering the zero-HUD principle and scene architecture.
- Add `ui-components` capability specifying the visual and functional states of shop objects (Register, Slips, Tools, etc.).
- Add `ui-visual-system` capability covering typography, color, materials, and lighting.
- Add `ui-interactions` capability covering input models, animation principles, and sound-visual coupling.

## Impact
- Affected specs: N/A (new capabilities)
- Affected code: Will guide the implementation of the `UIAgent` and frontend components.
